# -- Machine Learning Journey -- #

A structured collection of hands-on mini projects focused on building strong foundations in Machine Learning.

This repository emphasizes:
- Exploratory Data Analysis (EDA)
- Data Cleaning & Preprocessing
- Feature Engineering
- Model Building & Evaluation

Projects are organized progressively and include clear problem statements, structured notebooks, visualizations, and key insights.

Tools used:
Python • Pandas • NumPy • Matplotlib • Seaborn • Scikit-learn
